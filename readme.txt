The Crystal Ball plugin helps you monitor & log all changes and activities on your WordPress site. It will send log data to the Crystal Ball Insight website, so you can track all your site changes and display them over Crystal Ball Dashboard, Google Analytics, Google Ads, and Looker Studio.

== Description ==

<strong>Track your website changes and display them on Crystal Ball Insight</strong><br />

Once you activate your WordPress Web Changes on Crystal Ball, you can track all the site changes automatically. Changes of products and designs and securing your site by following all user activities. 

The plugin is lightning fast and works behind the scenes, so it doesn't affect site and admin performance.

This plugin solves the "know who did what and how affect your goals" by tracking who in your team makes a change and what changes, displaying it in an easy view on the Crystal Ball account or the Dashboard of your WordPress site.

<h3>With the Crystal Ball plugin, you can record:</h3>

* <strong>WordPress</strong> - Core Updates
* <strong>Plugins</strong> - Installed, Updated, Activated, Deactivated, Changed
* <strong>Pages</strong> - Created, Updated, Deleted
* <strong>Posts</strong> - Created, Updated, Deleted
* <strong>Custom Post Type</strong> - Created, Updated, Deleted
* <strong>Categories</strong> - Created, Edited, Deleted
* <strong>Tags</strong> - Created, Edited, Deleted
* <strong>Taxonomies</strong> - Created, Edited, Deleted
* <strong>Comments</strong> - Created, Approved, Unproved, Trashed, Untrashed, Spammed, Unspammed, Deleted
* <strong>Media</strong> - Uploaded, Edited, Deleted
* <strong>Users</strong> - Login, Logout, Login has failed, Update profile, Registered and Deleted
* <strong>Themes</strong> - Installed, Updated, Deleted, Activated, Changed (Editor and Customizer)
* <strong>Widgets</strong> - Added to a sidebar / Deleted from a sidebar, Order widgets
* <strong>Menus</strong> - A menu is being Created, Updated, Deleted
* <strong>Setting</strong> - General, Writing, Reading, Discussion, Media, Permalinks
* <strong>Options</strong> - Can be extend by east filter
* <strong>WooCommerce</strong> - Monitor all shop settings and options
* <strong>bbPress</strong> - Forums, Topics, Replies, Taxonomies, and other actions
* And more...

Crystal Ball Plugin is entirely FREE, and you can see all those changes in your WordPress Dashboard Settings. To get the User Website Changes in your Crystal Ball Dashboard and over Google Analytics Charts, you must insert your account's Crystal Ball API Key. 

== Installation ==

1. Install using WordPress' built-in "Add New Plugin" installer
2. Activate the plugin
3. Go to the plugin page (under Dashboard> Crystal Ball)
4. Insert your Crystal Ball API Key
